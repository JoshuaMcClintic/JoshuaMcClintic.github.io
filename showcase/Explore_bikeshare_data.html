Question 1¶

What are the most popular times for people to ride bikes?

The following cell takes the hour and minute data from the Start.Time column for each table and converts stores it as a decimal number (5:30pm turns to 17.5). This is done to convert the time data from discrete to continuous for use in visualizations. The Start.Time column in the Washington table outputs an error stating that the format of the column is ambiguous; using lubridate's ymd_hms() function tells the as.POSIXct() function what format the column is in; none of the other tables needed this.

In [6]:
# Create function to take Start.Time column and return decimal value for time
format_start_time <- function(data) {
    return (as.numeric(format(as.POSIXct(ymd_hms(data$Start.Time)), '%H')) + 
            (as.numeric(format(as.POSIXct(ymd_hms(data$Start.Time)), '%M'))/60))
}

The plots are actuall very similar for all three cities. However, the most interesting thing is that non-subscribers tend to bike more per trip than customers. This could be due to many reasons, though the most likely is economic effeciency. Because Motivate operates numerous bikeshare services across the US, it does not have a singular pricing plan, but we can use the pricing plan for one of the systems to generalize.

Chicago's Divvy has a few subscriber plans and one non-subscriber option. The non-sub option has a $\$$1 payment followed by $\$$0.20 per minute, while the subscriber plans offer three hours of free riding for $\$$36 ($0.20*60*3=36$) worth of riding for $\$$19.90 for one day. There is also an annual plan which offers $\$$3,285 worth of bike rides per year for only $\$$143.90 per year.

In general, these prices incentivize longer rides for non-subscribers to avoid paying the extra dollar per ride, and more, but shorter, rides for subscribers to avoid paying directly for the time on the bike. This is, in fact, what we see. The count of subscribers is around 4 times the count of the non-subscribers, while the time spent riding is larger for non-subscribers.
